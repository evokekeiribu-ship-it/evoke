---
description: 貼り付けられたテキストからギフト券番号のみを抽出し、LINEに送信する（金額なし）
---
ユーザーが送信したテキストデータの中から、ギフト券番号のみ（金額を含めず、コードのテキストのみ）を抽出し、その結果をPowerShellスクリプトを用いてLINEに自動送信してください。

手順：
1. 送信されたテキストから各行を解析し、以下の情報を抽出する。
   * **ギフト券番号**: `X` や英数字で構成される15〜16桁程度の文字列（例: `X1234567890ABCDE` や `0PX1-7U2M-GYDY-DK2D` など）。正規表現などを工夫して、コード部分だけを正確に抜き出すこと。
2. LINEに送信するメッセージを以下の形式の文字列としてまとめる。改行には `` `n ``（バッククォートとn）を使用すること。
   ```text
   【抽出結果】`nX1234567890ABCDE`nX0987654321FFEEE`n...
   ```
// turbo-all
3. 以下のPowerShellコマンドを実行して、作成したメッセージをLINEに送信する。`YOUR_MESSAGE_HERE` の部分を実際の抽出結果の文字列に置き換える（改行には `` `n `` を使用）。TokenとUserIdはそのまま使用する。

```powershell
$token = 'b64frT8W42B2iEfyL31QdZdw6s8tywPQGH9JBeE1VsyFaN7oPRh0w1rXAUiGLq75oZlWF9sQ0IK5r0emrw14HeIaZ/CudE5DcAQs9dceCLPf9sceuU3cj28NzG5ODveR/0rr4I5Kuowd5LyY45r70AdB04t89/1O/w1cDnyilFU='
$userId = 'U271592e02ec59829d590590021916744'
$msg = "YOUR_MESSAGE_HERE"
$headers = @{ 'Content-Type' = 'application/json'; 'Authorization' = "Bearer $token" }
$body = @{ to = $userId; messages = @( @{ type = 'text'; text = $msg } ) } | ConvertTo-Json -Depth 5 -Compress
Invoke-RestMethod -Uri 'https://api.line.me/v2/bot/message/push' -Method Post -Headers $headers -Body ([System.Text.Encoding]::UTF8.GetBytes($body))
```

4. 実行が成功したら、ユーザーに「金額なしで◯件のギフト券番号をLINEに送信しました」と完了報告をする。
