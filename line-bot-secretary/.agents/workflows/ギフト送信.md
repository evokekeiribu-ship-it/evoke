---
description: 貼り付けられたテキストからギフト券番号と額面を抽出し、合計金額とともにLINEに送信する
---
ユーザーが送信したテキストデータの中から、ギフト券番号（Appleギフトカードなどの16桁の英数字等）とその額面の金額をセットで抽出し、合計金額を計算した上で、その結果をPowerShellスクリプトを用いてLINEに自動送信してください。

手順：
1. 送信されたテキストから各行を解析し、以下の情報をペアで抽出する。
   * **ギフト券番号**: `X` で始まる16桁の英数字（例: X1234567890ABCDE）など
   * **額面**: ギフト券番号に近い位置にある金額（例: `10,000` や `10,000円` などからカンマや「円」を除外した数値）
   ※ もしテキストのフォーマットが表形式の場合、ギフト券番号と同じ行にある最も高い金額（販売価格ではなく額面）を取得するロジックを組むこと。
2. 抽出した額面の合計金額を計算する。
3. LINEに送信するメッセージを以下の形式の文字列としてまとめる。改行には `` `n ``（バッククォートとn）を使用すること。
   ```text
   【抽出結果】`nX1234567890ABCDE (10,000円)`nX0987654321FFEEE (5,000円)`n...`n`n合計金額: 15,000円
   ```
// turbo-all
4. 以下のPowerShellコマンドを実行して、作成したメッセージをLINEに送信する。`YOUR_MESSAGE_HERE` の部分を実際の抽出結果の文字列に置き換える（改行には `` `n `` を使用）。TokenとUserIdはそのまま使用する。

```powershell
$token = 'b64frT8W42B2iEfyL31QdZdw6s8tywPQGH9JBeE1VsyFaN7oPRh0w1rXAUiGLq75oZlWF9sQ0IK5r0emrw14HeIaZ/CudE5DcAQs9dceCLPf9sceuU3cj28NzG5ODveR/0rr4I5Kuowd5LyY45r70AdB04t89/1O/w1cDnyilFU='
$userId = 'U271592e02ec59829d590590021916744'
$msg = "YOUR_MESSAGE_HERE"
$headers = @{ 'Content-Type' = 'application/json'; 'Authorization' = "Bearer $token" }
$body = @{ to = $userId; messages = @( @{ type = 'text'; text = $msg } ) } | ConvertTo-Json -Depth 5 -Compress
Invoke-RestMethod -Uri 'https://api.line.me/v2/bot/message/push' -Method Post -Headers $headers -Body ([System.Text.Encoding]::UTF8.GetBytes($body))
```

5. 実行が成功したら、ユーザーに「抽出した◯件のギフト券番号（合計 ◯◯円）をLINEに送信しました」と完了報告をする。
